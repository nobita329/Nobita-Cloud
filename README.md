# ⚙️ Pterodactyl Auto Installer

One-command setup for Panel + Wings 🚀

## 🚀 Install
```bash
bash <(curl -s https://ptero.nobitahost.in)
```

## 📦 Includes
- Pterodactyl Panel
- Wings Daemon
- Docker Engine
- NGINX Web Server
- MySQL / MariaDB
- Redis Server

## 🧱 Requirements
- Ubuntu 20.04+
- Root access
- 2GB RAM (4GB recommended)
- Domain pointed to server

## ⚠️ Notes
- Open ports 80 & 443
- Use clean VPS
- SSL requires domain
